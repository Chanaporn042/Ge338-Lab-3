ภารกิจ 1 Training Strategy
1.1	ใช้ประเภทที่ดินทั้งหมด 5 Class ได้แก่ 1. Urban สิ่งปลูกสร้างต่าง ๆ 2. Agriculture พื้นที่เกษตรกรรม 3. Forest พื้นที่ป่าไม้ 4. Water พื้นที่น้ำ 5. Miscellaneous พื้นที่เบ็ดเตล็ด

1.2	 Training โดยการใช้ Baseline จาก ESA World cover

1.3	ใช้แบบ Train 80 Validation 20 

80/20 แบบสุ่ม vs Spatial Cross-validation ต่างกันอย่างไร?

-	ต่างกันคือ แบบสุ่ม 80/20 จะเป็นการสุ่ม โดยแบบให้ตัวโมเดลได้ฝึกฝนความแม่นยำจากข้อมูลอยู่ที่ 80% และทดสอบอีก 20% ส่วนตัว Cross-validation จากเป็นการแบ่งข้อมูลเพื่อนำไป train และ test 5 รอบ โดยแต่ละรอบจะแบ่งเป็น train 80% และ test 20% อีกทั้งตัว test จะสลับไปเรื่อย ๆ จบครบทุกตัว เพื่อลด Bias ของตัวข้อมูลให้มีความแม่นยำและถูกต้องมากขึ้น
  
1.4	ใช้ตัว Feature ของ B2, B3, B4, B8, B11, โดย 5 Feature นี้เป็นข้อมูลดิบของดาวเทียม Sentinel-2 อยู่แล้ว และใช้ NDVI เพื่อแยกค่าพืช NDWI เพื่อแยกน้ำ และ NDBI เพื่อแยกเมืองและสิ่งปลูกสร้าง

ภารกิจที่ 3 วิเคราะห์ Feature Importance
 
3.1 Band หรือ Index ใดมีความสำคัญมากที่สุด? — สอดคล้องกับความเป็นจริงทางกายภาพหรือไม่?

หากดูจากผลลัพธ์จะเห็นว่า Index ที่สำคัญคือ B4 (Red), B8 (NIR) และ NDVI โดยจะสะท้อนค่าพืชของพื้นที่ที่เป็นพืชและเกษตรออกมาเป็นสีเขียวและ Index NDBI ที่สะท้อนพื้นที่สิ่งปลูกสร้างต่าง ๆ ออกมาเป็นสีแดง ซึ่งหากดูจากการใช้ประโยชน์ที่ดินของพื้นที่นี้จะเห็นว่า ตรงพื้นที่ทางด้านตอนกลางไปถึงตะวันออกเป็นพื้นที่สีเขียวจริงตรงตามลักษณะทางกายภาพ และพื้นที่สีแดงที่เห็นสิ่งปลูกสร้างทางทิศตะวันตกเฉียงเหนือและใต้เป็นพื้นที่ของสิ่งปลูกสร้างจริงแต่อาจมีความคลาดเคลื่อนไปบ้างตามลักษณะทางกายภาพจริง และ NDWI ที่สามารถตรวจจับพื้นที่น้ำได้ตรงตามลักษณะทางกายภาพจริง

3.2 ถ้าตัด Feature ที่มีความสำคัญต่ำออก ความแม่นยำเปลี่ยนไปอย่างไร?

จากภาพที่ตัด B2, B3, B11 และ NDWI ออก จะเห็นว่าความแม่นยำลดลงโดยสักเกตเห็นว่าพื้นที่สีเหลืองซึ่งเป็น class เกษตรลดลงเล็กน้อย โดยเปลี่ยนไปเป็น พื้นที่สีเขียวแต่ก็ยังไม่สามารถตรวจสอบจากภาพที่ประมวลได้แน่ชัดว่าเป็นพื้นที่ป่า หรือพื้นที่เกษตร หากต้องเทียบกับกายภาพจริงจะยังขาดความแม่นยำจากการแปลภาพจริงอยู่

ภารกิจที่ 4 ประเมินความไม่แน่นอน

4.1 Class ใดถูกสับสนกันมากที่สุดใน Confusion Matrix? เพราะอะไรในทางกายภาพ?

Class Agriculture และ Class Forest เนื่องจากเป็นพืชเหมือนกันจึงอาจทำให้เกิดความสับสนในการจำแนกพื้นที่ดังกล่าวได้อีกทั้งบริเวณนี้มีพื้นที่เกษตรน้อยจึงทำให้โมเดลประมวลผลรวมกับพื้นที่ป่า

4.2 บริเวณที่โมเดลไม่แน่ใจ (Class Probability ต่ำหรือใกล้เคียงกัน) อยู่ที่ไหน?

จากภาพพบว่าพื้นที่ที่โมเดลมีความไม่แน่ใจคือพื้นที่สีดำโดยเฉพาะบริเวณที่ตะวันตกซึ่งในบริเวณเป็นพื้นที่ที่มีการใช้ประโยชน์หลายแบบโดยเฉพาะป่า และเกษตรที่อยู่ใกล้กันมากจนอาจทำให้โมเดลเกิดความไม่แน่ใจ

4.3 ถ้าต้องบอก Stakeholder ว่าแผนที่นี้เชื่อถือได้แค่ไหน จะพูดอย่างไร?

แผนที่นี้มีความแม่นยำประมาณ 85% ซึ่งถือว่าเชื่อถือได้ในระดับดีสำหรับการวิเคราะห์ภาพรวมของพื้นที่ เราได้ตรวจสอบความมั่นใจของผลจำแนกในแต่ละจุดด้วยโค้ด confidence map เพื่อแสดงว่าบางบริเวณอาจมีความไม่แน่นอนสูงกว่า การใช้แผนที่นี้ช่วยให้เข้าใจภาพรวมของการใช้ที่ดินและการเปลี่ยนแปลงภูมิประเทศได้ อีกทั้งยังเหมาะสำหรับใช้ในการวางแผนและวิเคราะห์ในระดับพื้นที่กว้าง เช่น การประเมินสภาพป่า พื้นที่เกษตร หรือเขตเมือง แต่สำหรับการตัดสินใจที่ต้องการความละเอียดมากกว่านี้ อาจต้องใช้ข้อมูลภาคสนามเพิ่มเติม และแนะนำว่าวรใช้ข้อมูลเพิ่มเติมประกอบด้วย
